Solidity编程（36）：storage，memory，calldata

发表于 2022-08-27 更新于 2022-08-29 分类于 Solidity Changyan：

solidity数据存储位置有三种：

storage：状态变量，永久保存在区块链上

memory：内存变量，只有函数运行中才存在，运行完就销毁

calldata：函数参数变量存储的位置，你可以理解为堆栈之类

默认条件下，智能合约的状态变量是storage，函数的临时变量是memory，但是可以通过关键字显式指定。

Solidity编程（33）：gas费

发表于 2022-08-27 更新于 2022-08-28 分类于 Solidity Changyan：

一次交易应该付多少燃油费？

由以下因素组成：

gas消耗：是交易消耗的gas量

gas价格：每单位gas需要多少eth

最终gas费 = gas消耗 * gas价格

gas价格更高的交易会被优先执行，多余的gas费用会被退还。

gas费有两个上限：

gas limit：由你设置的，最大gas费

block gas limit：由区块链网络决定的，区块的最大允许gas费

dotenv在浏览器无法使用

发表于 2022-08-27 分类于 Javascript, 待完成…… Changyan：

我们在react项目中使用dotenv，如果通过这种方式：

（1）通过npm install dotenv安装dotenv

（2）通过require('dotenv').config()来获取.env配置;

（3）通过process.env.MY_VALUE来读取参数

你会发现运行npm start时，会出一个错：

原因是react app是运行在浏览器的，而浏览器不支持fs，而dotenv却依赖于fs，同样，浏览器也是没有process.env的。

要怎么解决呢？

你无需安装引入dotenv，只需要完成以下步骤：

（1）在项目根目录下创建.env文件

（2）变量命名以REACT_APP_开头，这很重要！

（3）使用process.env.MY_VALUE来调用变量

（4）运行npm start即可
