可以采用Hardhat框架，框架内部有很多已经内置的库，可以在本地模拟一个真实的区块链网络，方便做测试。

Solidity编程（4）：查看交易gas费

发表于 2022-07-18 更新于 2022-07-21 分类于 Solidity Changyan：

我们知道，智能合约要提交到区块链上，需要矿工打包成区块，并帮我们运行，然后我们支付给矿工gas费用，所以这个gas费就是成本，太重要了。

那么支付的gas费用是多少，我们怎么查看呢，在代码里面又如何设置呢，这就是本文要写的内容了。

最开始，gas费采用的是竞价的方式；不过现在采用的是基础费+小费的方式。

我们直接去一个以太坊浏览器上查看，这个浏览器上可以查到所有部署在以太坊上的智能合约，交易情况，以及其他信息。

网址：https://etherscan.io/

这里我们在主页随意打开一项交易：

我一一解释一下：

from：是交易发起方地址

to：是交易接收方地址

Value：就是金额

所以这里的意思是，从from这个合约地址，到to这个合约地址，转账0.22…个以太坊，也就是328.87美金

关于gas的就是下面几项了：

Transaction Fee：交易gas费用

Gas Price：gas的价格

Gas Limit & Usage by Txn：最多能使用多少gas（设置的我们要给多少gas），本次交易使用的gas

Gas Fees：基础gas价格，最大gas价格（设置的gas最大价格），小费（也是设置的）

Burnt & Txn Savings Fees：烧掉的gas费，省下的gas费

它们的计算方式是这样：

交易gas费用 = 交易使用的gas X gas的价格

gas的价格 = 基础gas价格 + gas小费，如果超过最大gas价格，就采用最大值

烧掉的gas费 = 基础gas价格 X 交易使用的gas

省下的gas费 = （最大gas价格 - gas的价格）X 交易使用的gas

这里的基础gas价格是怎么来的呢？

因为我们的交易是在一个一个区块上进行的，同一个区块中，所有交易的基础费都是一样的，下一个区块中，基础费由上一个区块决定，有一点上下浮动。

比如我们的这个例子是在区块编号15184373上的，我们点击编号进入区块信息：

有一项Gas Used的状态，Gas Target是它的载重，

这个值会影响下一个区块的基础gas费：

如果+100%，那么下一个区块的基础费会增加大概12.5%

如果+10%，那么下一个区块的基础费会增加大概1.25%

如果-10%，那么下一个区块的基础费会减少大概1.25%

这个多算算大概就清楚了，因为也改变不了什么，了解就行。

元宇宙初探：作为程序员，我是怎么看待元宇宙的？

发表于 2022-07-18 更新于 2022-08-25 分类于 元宇宙 Changyan：

现在已经晚上11点多了，我才开始写文章，实在是有些匆忙。

并不是我有意这样，而是今天一天时间下来，我只在做一件事，那就是配置部署环境，到现在，还没配完……

事情是这样的：

昨天有个朋友让我帮忙做一个小程序，很传统的小工具，所以我就用很传统的方式来搭，没办法朋友的忙也要帮一帮。

你以为程序员最烦的事情是什么？不是复杂的代码，而是配置环境，每种工具都有版本，不小心版本还不兼容，所以配置环境就是最烦了。

因为我也有很长一段时间没写代码了，也没有属于自己的环境（一般程序员都有一套他自己很熟悉的环境，该有的安装包估计也在他的硬盘里，搭起来虽然麻烦，但是也会一气呵成）

我重新拾起，所有的工具自然是要用新的，不能用以前的旧版本，所以一天时间里，要么处理繁琐的步骤，要么不小心版本不兼容，要重新安装，或者出了一点小错，要去找bug，要么被小问题弄烦了，出去散散心。

前段时间我虽然有发布一些代码，但其实都是web3相关，没有环境问题；web3底层是区块链技术，环境相关的东西封在一些智能合约里，智能合约是服务与服务的交互，引用什么直接import，所以是完全专注于逻辑，配置环境很少很少。

猛然的从web3到web2，就感觉从城里到了乡下，干啥啥不方便，吃个饭还要拾柴火，挑水，热灶，忙活半天也没忙活到米下锅，那种要做一些原始的事情的感觉。

现在，虽然我已经配置的差不多了，就是会出一些小问题，等着我一个一个解决，文章匆匆写成。

真是由奢入简难。

之前我总是说元宇宙是趋势什么的，都是站在对我们工作和生活的改变上，而站在程序员的角色上，我依然是这么认为的。

杰彬老师说过：

看一件事情是否长久，就看一个指标：

这件事是不是真的带来了价值？

比如：

之前买卖，是一层一层代理最后到达用户

现在团购，是厂家出来的货，直接触达用户

这个发展是带来了价值的，因为它降低了商品触达用户的成本，省下了层层代理要分走的利润

那么站在程序员的角度，元宇宙编程带来的价值是什么呢？

之前写程序各种标准不一，环境不兼容，配置环境，学习标准成本很高

以后标准只有区块链技术，服务封装成模块引用，各自为各自负责，省去配置环境，节省了时间成本

这样看来，肯定是带来了价值的

所以你认为它是不是趋势呢？

元宇宙初探：这是一个元宇宙的故事

发表于 2022-07-18 更新于 2022-08-25 分类于 元宇宙 Changyan：

今天说一说我们的物理宇宙，还有元宇宙：

我们的信息都被记录下来

我们在这个宇宙中，不管是和别人一起活动，还是我们独处，我们的信息都是被一一记录下来了的。

很多人说，人在做，天在看，我一向不相信这么没有逻辑的事情，但是它背后也有一些东西。

当然不会有一个所谓的“上帝”，拿着笔，记录每一个人的言行，那这个“上帝”也太闲了点，估计也忙不过来。

如果我们的信息要被记录下来，一定是分布式，一个个区块链节点记录下来的，这才是更现实的一种方式，这就是为什么元宇宙能做到把所有信息记录下来的原因，因为元宇宙使用区块链，能够实现分布式。

那么我们物理宇宙中，记录我们的信息，也是分布式记录的，大自然的很多东西都是天然的硬盘，比如树木，石头等等。

物理宇宙做到了，只要世界上还有树木山石等等的存在，历史就会一直被记录，被延续。

每一个地方，曾经发生过什么，都是被记录在在场的硬盘里，只是不是所有人都有能力读取大自然硬盘里存储的东西。

为什么我们从没去过古代，考古学家们也能大致描摹出中国几千年的历史呢，就是因为考古学家们就是专业解读大自然硬盘的人们，只不过这些人，读取硬盘的能力也是有限，所以他们了解到的信息也有限。

真正能把这些硬盘的东西事无巨细的读取下来，我相信是有人能做到的，但今天要讨论的不是这个问题。

今天要说的是元宇宙，那么对于元宇宙，记录下我们信息的是一个一个区块链节点，元宇宙做到了只要世界上还有电脑存在，元宇宙就会存在。

只是元宇宙和物理宇宙不同，就是读取，物理宇宙信息的读取太艰难，能读取的人太少了，但是元宇宙不一样，只要对方授权，你就可以通过点击一下鼠标，读取到所有信息，事无巨细。

正是因为物理宇宙不易读取的特性，如果一幅达芬奇的画，是原作的话，上面必然有原作的信息，只是读取的人不会读，常常分不清真品赝品；

也正是因为元宇宙的易读取的特性，如果一幅画，能够通过点一下鼠标就追溯到它的原作者，让赝品变得不可能，让原创者充分受益。

我们为自己的行为负责

在物理宇宙中，常常说的一句，因果报应，我也一向不相信这么没逻辑的事情。

虽然我们的行为确实被大自然的硬盘记录下来了，但是没人能读取呀，追溯不了，怎么谈因果报应。

如果真要做到这一点，那肯定要找到一个非常牛的人，他能够读取大自然的信息，然后让他去一个一个读每一个人的信息，再根据他做的好事，坏事给予奖励和惩罚，这个太不现实了。

首先这个人你找不找得到，其次，你找到了，让他一个一个算账，这么多地球人，只怕算到人都已经入了黄土才算的完。

但是，如果在元宇宙，就可以轻松做到这一点：

每个人的信息是被记录下来了的

你可以轻松读取到每个人的信息，当然是在对方授权情况下

最重要的是，可以通过编程，实时对数据进行处理，也就是在数据产生之初就可以做裁断，因果报应瞬间进行

所以，就能做到这样：

法庭要求追溯这件事的前因后果，于是写一段代码，一键追溯，立马把所有相关人员都能找出来，谁谁谁在什么时候干了什么坏事

法庭可以很快裁决，谁谁谁该受罚，谁谁谁该受奖

当然，也能做到这样：

如果我们做了坏事，这个坏事在当下就可以立刻辨别出来，比如网暴了一个人

于是可以立刻追溯到，谁谁谁干了什么坏事，很快接受惩罚

当然，也可以做到这样：

如果我们做了好事，N年后，你帮助的那个人暴富了

然后这个人回忆起当初帮助过他的人，想要报恩，怎么找

立刻追溯，找到当年什么时候给予过自己恩惠的人，不管他在天涯海角，你都可以报恩

当然，也可以做到这样：

你现在在一个平台活动，为平台做的贡献平台能通过发代币体现

N年以后，平台爆发了，成为top平台，价值一下子到了千亿

于是，当初平台发出来的代币价格也暴涨，如果你的代币一直没卖，可能这个时候你就靠这些代币变得有钱了

数字资产属于自己,资源随用随取

在我们还在农村的时候，其实就是能够做到随用随取。

比如这一餐想吃茄子，那么去地里摘两个茄子，回来炒一碗也就可以了，不会说，只想吃茄子，非要把丝瓜都摘回来，放在家里变得不新鲜。

比如我们呼吸，也是需要多少氧气就呼吸多少，不会说还要把氧气存储起来明天用。

但很多不自然的事情，我们没有做到随用随取

比如，我们使用网络资源，在抖音刷视频，使用淘宝来买东西，不管是刷的动作，还是买东西的动作，我们都是没有付出的，因为这是平台在付出。

为此，我们也要付出一些代价，比如我们的数据资产被平台占有，产生的价值也归平台所有。

但是，有给我们这个选项吗，我们用多少网络资源就付费多少网络资源，我们的数据资产依然是属于我们自己，没有。

因为这个互联网没有做到让我们每个人为自己的行为负责，所以也就无法做到让数据资产回归个人

在元宇宙就可以做到这一点：

智能合约是部署到区块链上

我们只要需要在区块链做修改的操作，都需要为这次动作付费

比如，我们购买NFT，要付gas费，要买币，付gas费

付费是付给帮我们执行这次智能合约的挖矿者

网络资源分布在全球各地，各个区块链节点挖矿者承担着，他们为他们每次执行智能合约得到一定费用，非常合理，由发起这次智能合约的人付费，也非常合理。

各自承担各自的职责，同时数据资产属于每个人自己。

网络资源随用随取，使用一次就付一次费，不使用就不付费，也非常合理。

这一部分没有写得太清楚，如果你看懂了，那你非常厉害

所以，元宇宙是一个多么迷人的世界呀，估计每个了解它的人都会向往，它会让人变得更善良，让人与人之间的交往变得真诚，让每个人做自己的主人，等大家都了解它的好以后，成为趋势是毋庸置疑的事情。
