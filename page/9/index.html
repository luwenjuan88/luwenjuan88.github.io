NFT探秘：NFT是什么？

发表于 2022-07-02 更新于 2022-08-25 分类于 NFT Changyan：

如果你是站在未来的某一天，你肯定不会问这个问题；因为那个时候，NFT就是我们手边的任何东西，只是现在无法理解。

先看看NFT的定义：

NFT是非同质化代币，拥有唯一性，不可篡改性，可确权性；

这个定义用人话来说，就是元宇宙的资产，当然到元宇宙形成的那一天看，你就知道了元宇宙资产是个什么情况。

现在呢，我只能用物理世界的资产类比，来帮助你理解。

物理世界的资产有哪些呢？

纸巾，柴米油盐，这些是会消耗的资产

衣服，鞋子，车子，这些是会折旧的资产

房子，古董，这些是能增值的资产

商标，网址，这些IP类资产

书籍，音像，这些版权类资产

在元宇宙世界，目前暂时还没有会消耗的资产，但是我觉得也会出现，就是不知道会不会以NFT的一个分支出现，还是有一个新的名称，但这不重要，就是个名字的问题。

所以目前的NFT就是另外几种资产了，这些资产有一个特点，就是可以交易（俗称卖钱）。

物理世界的资产是唯一的：

不管是你的什么资产，其实都是唯一的（他不可能点一下copy键复制无数个），也是不可篡改的（不可篡改的意思是它的本质发生篡改，让他的价值发生变化，任何已经发生在其上的事情都会被岁月记录下来），他也是可以确定是你的，你有处置权，除非被人偷了；

这里我们再说一下互联网，为什么互联网无法成为一个宇宙，就是因为他无法做到唯一性，不可篡改性，和可确权性（这是完整宇宙必须具备的底层逻辑），怎么说呢？

唯一性：一个图片，可以被copy无数次，而且没人记录它的行为，copy之后，你无法确认哪个才是最初那个，因为在硬盘上，存的就是01序列，谁能从同样的01序列中，确定哪个才是原始的呢；

不可篡改性：更不可能了，想怎么改就怎么改，而且没人记录你改动的行为，你改不爽了，还可以穿越回去（改回过去的版本）

可确权性：01序列谁拿到就是谁的，怎么确权？

所以说互联网世界无法成为一个完整的宇宙，因为无法解决这三个底层问题，正是因为这样，互联网世界充满了虚假的繁华，盗版横行，你看到的小姐姐有可能是个抠脚大汉，这就不是一个完美的虚拟世界。

为什么元宇宙能出现呢，就是因为从技术上解决了这个问题，从区块链技术成为元宇宙的基石起，元宇宙就天然带有唯一性，不可篡改性，和可确权性。

虚拟宇宙原本就想要成为一个世界，只是技术一直限制了发展。但自从解决了这个问题，这样一个空旷的元宇宙出现以后，在上面就可以自然生长出一系列现实宇宙存在的逻辑，NFT就是其中一个。

这也是为什么我说元宇宙是趋势的原因，是大家共同的需求也好，是虚拟宇宙本身的力量也好，它突破以后，必然会迅猛发展。

接着上面说……

虚拟世界的资产NFT拥有了现实世界资产的属性，能够在虚拟世界承载作为资产的责任，但它远远不止于此，它有比现实资产更加强大的能力，这是虚拟世界赋予它的。

它不会折旧：我们的衣服，穿着穿着就旧了对吧，但是虚拟世界，它也不会旧呀

它的流通性更强：我们想要卖掉元宇宙的资产，直接要到对方钱包地址，转过去就行了，但是我们要卖掉现实世界的资产，我们还需要找物流，使劲讲解，成本很高

可确权力更强：现实世界的资产是可确权的，但是一旦有了假货，要找专业的鉴定师，还不一定准，资产如果自己有记忆，肯定知道自己是真的，但我们很难知道；NFT就没这个烦恼了，只要一追溯，就能看到它所有的历史，确权分分钟的事

它可以承载更多：现实世界的资产，因为受物理的限制，他能承载的很难超越它的物理组织界限，但是元宇宙不会，它是靠代码，理论上，所有的资产，只要你愿意，他能够满足你所有需求，比如一个钱包，不止是存钱，你需要的话，它还能进行资产分析，也就是可以赋能。

所以说，元宇宙的资产具有更加丰富的属性。

所以说，NFT远远不是一张图片，那一张图片只是我们认识NFT的一个切入口。

目前有两类NFT：

一类是艺术品：这一类是艺术家或者机构在参与

一类是IP：这一类是运营方在参与

他们都跟图片有关，所以大家自然而然觉得NFT就是图片，但其实，NFT可以是任何东西，他可以是：

基础设施：我们现实世界的各种东西

制度：一个公司的制度，一个组织的结构等等

目前的图片只是小小的一部分，因为元宇宙的大火，很多人趁着热度在其中炒作圈钱，炒作NFT的概念，让人觉得这是一场投资活动，想想确实是投资活动，但是不是投机，今天买，明天卖

而是长远考虑，今天买，未来我在元宇宙中是不是会用到，再就是未来他会不会在元宇宙被人认可它的价值。

所以，我今天就在想购买NFT数字藏品的几个思路，时间不够了，随便写写吧：

1.能够为你目前做的事情赋能
比如旅行博主可以购买一些景点的数藏，然后在旅游的时候展示，吸引一波元宇宙玩家

今天我还在想，书是不是也该做成NFT了，毕竟在互联网，电子书随便复制，因为你购买了一本电子书，这本电子书并不是你的，你并不能说再把它卖出去（你卖的时候他就已经成盗版了，谁知道你卖的是不是买的时候那个电子书呢），只是购买了一个观看权。

但是，如果它是一个NFT，你就是可以卖，因为可以追溯到该NFT的购买记录，只要它在那里，说明你就是在官方买的。

NFT初探：批量生成NFT，素材要怎么制作？

发表于 2022-07-02 更新于 2022-08-25 分类于 NFT Changyan：

本文是针对想要进入NFT领域的设计者，以及艺术家。不过我也想告诉更多艺术家们，元宇宙是趋势，迟早都是要进入这个领域的，不妨早点学习，也欢迎交流。

先看个案例

我们先看一个案例，这个是NFT中国的中华小橘猫：

1.体态固定

我们可以看到，每一只猫的体态都是一样的，也就是大家都是同样的身体，都是向左歪着。

为什么要体态一样呢？因为体态决定了其他素材的位置，比如这样的体态，眼睛就是长在固定位置的，帽子也是在固定位置的。

每个素材位置固定，才能在左排列组合的时候不会出现混乱，比如眼睛长在鼻子的位置，或者嘴巴跑出了脸。

当然，体态固定，颜色可以改变，比如这只猫的体态从黄色换成绿色，也没有问题。

2.素材分组

比如这只猫，它的体态只有一个，所以体态的素材数量是1。

从我截图的里面，可以看到至少有4种不同的帽子，假设帽子素材数量是10。

从我的截图里面，可以看到有4种不同的眼睛，假设眼睛素材数量是4。

从我的截图里面，可以看到嘴巴有两种，假设嘴巴素材数量是2。

从我的截图里面，可以看到嘴巴上有装饰，口罩，花，耳机，三种，假设嘴部装饰数量是3。

从我的截图里面，可以看到有4种衣服，假设衣服的数量也是10。

那么如果只有这些素材，那么可以最多生成多少张图片呢？

1 X 10 X 4 X 2 X 3 X 10 = 2400张图片

因为中华小橘猫的素材分组有很多，所以这只猫形态各异，显得都非常有特点，如果素材分组少，可能会显得更加单一。

PS里面怎么制作素材

其实很简单，先画身体。

体态固定

在画完身体后，这个体态就要保持固定了。

如果只需要一种颜色的身体，那么身体就不需要再做更多素材，但是如果需要不同颜色的身体，那么可以做不同颜色身体的素材。

比如我做的猫脸表情包，就做了几个颜色的猫脸，黑色，红色，和白色：

不同素材用不同图层

比如三种颜色的脸，我用的是三个图层，当然再画的时候，我会隐藏掉一些图层，避免干扰，但是图层还是要画的。

比如画眼睛时，当我画完第一种眼睛时，就隐藏图层，再新建图层画第二种眼睛，一共画了4种眼睛。

当然其他素材也是这样，比如嘴巴有3种：

NFT初探：选择NFT的三个标准

发表于 2022-07-02 更新于 2022-08-25 分类于 NFT Changyan：

现在NFT炒作挺厉害的，很多人根本没有选择NFT的意识，而是见到首发就抢，抢完就抛的状态。

但是如果你有了选择的意识，你至少是走在了前面了。

这里说说三个标准：

标准一：制作精美，有艺术价值

就像现实世界中的艺术画作一样，制作精美的自然会更有收藏价值。

而现在制作精美的也更容易受到藏家的关注，一旦关注到位了，它也容易在早期形成共识，所以越早期投资NFT的，越容易得到更大回报。

它的主要作用是收藏价值，同时也有商业价值，它的商业价值有两个方面：

一是收藏同一系列NFT的藏家，可以形成圈子，这个作品直接把这样的圈子筛选出来了，大家有共同属性，容易产生商业价值

二是未来万一作者火了，他的作品价值是不可估量的，所以这个需要眼光和运气

所以说，你收藏艺术品，第一目的一定不是它的商业价值，否则你就失去了欣赏艺术品的那颗心了，也就找不到自己真正喜欢的作品。

它的商业价值，你要当作附赠的一个彩蛋而已。

标准二：具有共识，有商业价值

如果你单纯考虑商业价值，那就要用商业的眼光来看待。

什么是共识呢？就是大部分人认可的标签。

比如加密朋克，是NFT的鼻祖，你不认也得认，那个位置已经被它占了，这个已经成为世界的共识，所以加密朋克一张头像能卖到几百万。

一般什么是具有共识的呢？

行业第一：比如加密朋克，比如比特币是首个数字币

在物理宇宙中已经存在共识：比如茅台的股票代码域名被炒到几十万

标准三：团队强大，有发展潜力

不得不说，NFT好不好，还是要靠团队运作。

好的NFT，除了制作精美，大家有共识之外，还有就是团队靠不靠谱。

因为再好的东西，虎头蛇尾，团队不干了，烂尾了，一样的一文不值，同时还会消耗用户的信任。

我想说一个好的团队，至少要看这些：

他们的未来发展规划如何，是否有按照规划一步一步按序执行

他们是如何考虑团队的利益的，他们是如何考虑用户利益的

他们的发展规划是否有逻辑，还是照搬别人，有没有危机预案

我想说的

其实选择NFT是跟你的目的相关，也和你的行业相关，不一定只是买下来等它升值。

比如说李宁买下无聊猿猴，其实是看中无聊猿俱乐部的粉丝群体，让无聊猿穿上品牌的衣服，可以助力自己的品牌推广。

很多传统企业，买下NFT，也是为了参与到元宇宙的运作，想要在元宇宙找到品牌推广破局的方案。

所以在选择的时候，就要思考：

购买的NFT背后的粉丝群体

购买的NFT背后的文化生态与品牌的匹配度

购买的NFT对于品牌是否有更多的赋能，有利于品牌找到新的模式

Discord安装

Discord原本是给一些玩游戏的人组队连麦用的，后来由于它功能的强大，成为NFT项目方沉淀支持者，与支持者和粉丝交流的地方。

Discord网址：https://discord.com/

打开Discord网址（需要科学上网），注册账号，然后跟随它的指引一步一步来就行。

记得验证手机的时候，前缀是中国+86，另外还会验证邮箱，中途可能会验证失败，没关系，多尝试几次就行了。

Discord设置

开启双重验证

这里需要安装谷歌身份验证器

苹果手机搜索（需要美国账号）：Google Authenticator
安卓手机在谷歌play上搜索：Google身份验证器

即可安装

用谷歌身份验证器扫码,填入验证码即可

开启SMS验证

就是手机验证码的验证，输入密码即可开启。

关闭私信

在隐私设置里关闭私信，不允许任何人给你发私信，也不要点击任何人给你发的链接。

也不允许任何人加你好友，这是对初次使用软件的建议，保护你网上冲浪的安全。

加入社区

在各大项目的主页，可以看到加入Discord的邀请，点击左侧加号，加入服务器，输入邀请链接即可加入某个社区。

也可以搜索公开的社区加入。
